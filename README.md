# Лабораторная 2 по дисциплине "Технологии программирования"

## Постановка задачи:
1) Магазин детских игрушек. Покупатель может указать промокод, при введении которого цены снижаются на 5, 10 или 15% (в зависимости от типа промокода);
2) Составление модульных тестов к проекту.

## Краткое описание проекта:
Веб-магазин, разработанный на фреймворке Django. Пользователь может просматривать список товаров, переходить на страницу оформления покупки и завершать заказ, указав имя, адрес и при желании промокод на скидку.

## Используемый язык:
Python 3.8.

## Используемые технологии:
1) Фреймворк - Django;
2) БД - PostgreSQL;
3) Тестирование	- Django Test Framework;
4) Архитектура - MVC;

## Вывод
В ходе выполнения работы был создан веб-приложение на Django с использованием MVC. Проект продемонстрировал обработку пользовательского ввода, взаимодействие с базой данных и выполнение логики (расчёт цены по промокоду).
